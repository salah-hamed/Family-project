📦 إنشاء قاعدة البيانات

افتحي Firebase Console:

فتح Firebase
  • اضغطي Create Project
  • اكتبي اسم المشروع
  • اضغطي Continue
  • انتظري إنشاء المشروع
Firebase هو المكان اللي هيحفظ طلبات العملاء تلقائي ✨

🗂️ إنشاء Firestore

  • ادخلي Firestore Database
  • اضغطي Create Database
  • اختاري Start in test mode
  • اضغطي Enable

دلوقتي هننشئ Collection لحفظ الطلبات.

اسم الـ Collection المقترح: orders

واكتبي هذا البرومبت لـ ChatGPT:

اقترح أسماء Collections و Documents لتطبيق توصيل قهوة يحتوي على طلبات وعملاء ومندوبين

🔑 الحصول على Firebase Config

  • ادخلي Project Settings
  • اختاري Web App
  • اكتبي اسم التطبيق
  • اضغطي Register App

هيظهر لك Firebase Config.

انسخي: apiKey projectId appId وكل البيانات الموجودة ✨

💻 تعديل الكود وربطه بقاعدة البيانات

هذا هو Firebase Config الخاص بي: ضعه داخل الكود واربط نموذج الطلبات بFirestore بحيث يتم حفظ كل طلب تلقائي
